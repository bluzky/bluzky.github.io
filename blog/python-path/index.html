Bài viết này liệt kê một số hàm thường dùng để làm việc với đường dẫn trong Python Các ví dụ trong bài sử dụng môi trường hệ điều hành Ubuntu

1 Lấy đường dẫn thư mục, file name và extension

  1. Lấy đường dẫn thư mục
import os.path
print os.path.dirname("\etc\host.conf")

Link document

  1. Lấy tên file
import os.path
print os.path.basename("\etc\host.conf")

#host.conf

Document

  1. Lấy đường dẫn thư mục và tên file
    import os.path
    dir_path, file_name = os.path.split("\etc\host.conf")
    print dir_path #\etc
    print file_name #host.conf
  • Hàm os.path.split trả về 1 tuple gồm 2 phần, phần tử đầu là đường dẫn thư mục, phần tử thứ 2 là file name
  • Nó không tự nhận biết đường dẫn thư mục và đường dẫn file.

  • Document

  • Lấy extension từ đường dẫn hoặc tên file

  import os.path
  text, extension = os.path.splitext("\etc\host.conf")
  print text
  print extension
  • Hàm os.path.splitext trả về 1 tuple gồm 2 phần: phần tử đầu là đường dẫn truyền vào sau khi đã bỏ đi extension, phần tử thứ 2 là extension (bao gồm cả dấu . )

2 Kiểm tra đường dẫn tồn tại

-

import os.path
print os.path.exists("/etc") # True
print os.path.exists("/boot/some_random_file") # False
  • Hàm os.path.exists làm việc với cả đường dẫn file và thư mục
  • Document

3 Lấy đường dẫn tương đối

  • Hàm os.path.relpath(path, dir) trả về đường dẫn tương đối của path so với dir.
  • Hàm này không kiểm tra sự tồn tại của path và dir
import os.path
print os.path.relpath("/home/aa/bb/ff.html", "/home/aa/") # bb/ff.html
  • Document

4. Lấy đường dẫn tuyệt đối

  • Hàm os.path.abspath(path) trả về đường dẫn tuyệt đối của đường dẫn có các thành phần tương đối
import os.path
file_path = "/usr/bin/../bin/vi"

# loại bỏ ../
print os.path.normpath(file_path)
# /usr/bin/vi

# tương tự như os.path.normpath
print os.path.abspath(file_path)
# /usr/
  • Document

5. Lấy đường dẫn thực từ symbolic link

  • Symbolic link là một loại file đặc biệt, chứa tham chiếu tới file khác dưới dạng đường dẫn tuyệt đối hay tương đối. Ví dụ: "Short cut" trên Windows cũng có thể coi là một dạng Symbolic link
  • Hàm os.path.realpath(path) trả về đường dẫn thực, sau khi xử lý xong symbolic link
import os.path
print os.path.realpath("/usr/bin/vi")
#/usr/bin/vim.tiny
print os.path.realpath("/usr/bin/../bin/vi")
#/usr/bin/vim.tiny

6. Lấy đường dẫn tới interpreter, script đang thực thi

  • sys.executable chứa đường dẫn tới interpreter
import sys
print sys.executable
#/usr/bin/python (trên HDH Ubuntu)
  • Lấy đường dẫn tới file script đang được chạy
import os.path
print os.path.realpath(__file__)
